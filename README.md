# FinanceApp
FinanceApp

FinanceApp es una aplicación móvil desarrollada en Kotlin utilizando Jetpack Compose como framework de interfaz gráfica. Su propósito es ayudar a los usuarios a gestionar sus ingresos y gastos, ofreciendo una manera sencilla y segura de llevar el control de sus finanzas personales.

 Características

Registro y autenticación de usuarios mediante Firebase Authentication.

Validación de correo electrónico al momento del registro.

Inicio de sesión seguro con credenciales almacenadas en Firebase.

Gestión de ingresos y gastos:

Añadir nuevos movimientos.

Eliminar registros cuando el usuario lo desee.

Almacenamiento de datos en Firebase, lo que garantiza seguridad y disponibilidad de la información.

Interfaz moderna y reactiva construida con Jetpack Compose.

Explicación de la aplicación

FinanceApp fue diseñada para que cualquier persona pueda gestionar fácilmente su información financiera.
La aplicación permite:

Registrarse utilizando un correo electrónico válido, que será verificado mediante Firebase.

Iniciar sesión con las credenciales previamente registradas.

Registrar ingresos y gastos, cada uno con su descripción, monto y fecha.

Visualizar todos los movimientos financieros almacenados en Firebase.

Eliminar registros que el usuario ya no desee mantener.

Mantener la información sincronizada y disponible desde cualquier dispositivo.

 Tecnologías utilizadas

Kotlin – Lenguaje principal de desarrollo.

Jetpack Compose – UI declarativa para Android.

Firebase Authentication – Registro, inicio de sesión y validación de usuarios.

Firebase Firestore / Realtime Database – Almacenamiento de datos financieros.

Android Studio – Entorno de desarrollo.
